# SUPPORT.md

## English

Use GitHub issues for bugs and feature requests.

Include:

- the exact steps you took inside Praxis (button setup, command, key sequence)
- your Praxis app version (Help / About) and OS
- a sanitized reproduction path or sample input
- the output or behavior you expected, and what you saw

Do not post private paths, customer reports, proprietary binaries, tokens, or exploit details in public issues.

## 日本語

バグや機能要望は GitHub issue を使ってください。

含めてほしい情報:

- Praxis 内での具体的手順（ボタン構成、コマンド、キー操作）
- Praxis アプリのバージョン（ヘルプ / バージョン情報）と OS
- 匿名化した再現手順またはサンプル入力
- 期待した挙動と実際の挙動

public issue に private なパス、顧客レポート、プロプライエタリなバイナリ、トークン、攻撃詳細を載せないでください。
